## Tech Stack tổng hợp

| Thành phần | Công nghệ |
|---|---|
| Runtime / Ngôn ngữ | Node.js + TypeScript |
| Framework | Express.js |
| Database | Supabase (PostgreSQL) |
| ORM | Prisma |
| Cache / Cart | Redis |
| Validation | Zod |
| Auth | JWT (Access 15p + Refresh 7 ngày) + Passport.js (Google OAuth) |
| File Storage | Cloudinary |
| Email | Nodemailer + Gmail SMTP |
| SMS | Twilio |
| Payment | Static QR chuyển khoản ngân hàng (ảnh tĩnh, không cần cổng thanh toán) |
| API Docs | Swagger (swagger-ui-express) |
| Frontend | Angular |
| Analytics | GA4 + Microsoft Clarity (client-side) + endpoint batch nội bộ |
